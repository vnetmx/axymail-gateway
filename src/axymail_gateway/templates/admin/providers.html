How to use

Each provider maps one Google Cloud OAuth app to a URL slug. Users go through the consent flow once per account and get back a bearer token.

Authorize URL GET /v1/oauth/{name}/authorize → returns {"auth_url": "…"}
Callback GET /v1/oauth/{name}/callback?code=… → returns {"account_id","token","email"}
After that Use the token as Authorization: Bearer <token> on all IMAP/SMTP endpoints
